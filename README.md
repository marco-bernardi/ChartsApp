<h1 align="center">ChartsApp</h1>
<p>
  <h2 align="left"> A simple chart maker app</h2>
  ChartsApp is a professional desktop application for plot charts. ChartsApp's purpose is to be easy to use, stable, light and accurate. ChartsApp is based on free and open source software, providing the basic tools that users have come to expect from a modern charting application. The software offers features such as editing datasets, saving edits, export charts and export to csv, zoom and navigation on charts.
